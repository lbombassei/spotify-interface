# UI Clone Spotify

## Why clone?

- Created this project for learning purposes so I can get a feeling for tailwind Css and Next JS With React


## Stacks Used:

- NextJS
- React
- TypeScript
- TailwindCSS
- NPM


## To run on your local machine:

- Clone repository
- Execute: `npm install`
- Execute: `npm run dev`
- Go to: `http://localhost:3000` to see the website online
